# java-movie-precourse
구현 해야할 기능적 요구사항

영화 클래스
    객체에 들어갈 정보
        영화이름
        영화 상영 길이

영화 상영 시간 클래스
    영화 이름
    영화가 시작하는 시간 및 날짜

예매 시스템 클래스
    -영화 선택 ( 여러 영화 선택가능 )
    -시간이 겹치면 불가
    -영화, 시간 고른 뒤, 좌석 선택
    -예약된 좌석은 다시 선택 불가

결제 시스템 클래스
    -특정 조건 성립 -> 추가 혜택
    -포인트 사용 가능
    -결제 수단에 따른 추가 할인

좌석 클래스
    -행(알파벳)과 열(숫자)
    -등급에 따라 가격 다름
        S -> 18000원
        A -> 15000원
        B -> 12000원

특정 조건 클래스 (각각 클래스로 생성?)
    1. 무비데이 조건
        매월 10,20,30일 -> 10% 할인
    2. 시간 조건
        오전 11시 이전 or 오후 8시 이후 -> 2000원 할인
    - 동시 적용 가능, 1->2 순서로 적용
    3. 결제 수단에 따른 할인 클래스
        1. 신용카드 결제 -> 5%할인
        2. 현금 결제 -> 2%할인

포인트 클래스
    - 정해진 것 없음
        결제가격의 1% 적립 (임의)
